# Find the error in FPA implementation

The FPA implementation on the [pymetaheuristics](https://github.com/Valdecy/pyMetaheuristic/tree/main) library has an error. This error affects the convergence time of the method.

You must investigate the code and produce a Python notebook pointing where the error is, explain why it affects convergence, and provide a solution to the error.

Please send your Notebook file (.ipynb) in SIGAA. The assignment can be done in pairs.

For the regular students, the first pair of students to send a correct solution through SIGAA will receive A. The other students that send the correct solution will receive B.

For the students that are enrolled as an isolated course, the first pair of students to send a correct solution by mail will receive A. The other students that send the correct solution will receive B.
